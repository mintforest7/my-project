# Style Rush

A complete 2D fashion competition prototype made with Python and Pygame.

## Run

```bash
cd style_rush
pip install -r requirements.txt
python main.py
```

## How To Play

- Click **Play** to receive a random theme.
- Pick clothes from hair, tops, bottoms, dresses, shoes, and accessories.
- Use **Preview** when you are ready, or wait for the timer.
- AI judges score the outfit based on theme tags, colors, and style variety.
- Earn coins and XP, then unlock more clothes in the shop.

The pixel-art clothes are placeholder shapes drawn in code. Replace the drawing functions later with real images if you want custom art.
